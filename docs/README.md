# AEMS Heritage Map — Offa, Nigeria

Interactive map of cultural heritage assets documented by the Offa Heritage Documentation Initiative.

**[Open the Interactive Map](index.html)**

## About

This map was automatically generated by the AEMS Heritage Pipeline, processing 500+ community photographs from Wikimedia Commons and geocoding them using a specialized local gazetteer.

**Key metrics:**
- 321 heritage assets geocoded (64.2% success rate)
- 40 unique locations identified
- 100% offline processing (no external API dependency)

## Legend

- 🟢 **Green**: Exact match
- 🔵 **Blue**: Partial match
- 🟠 **Orange**: Fuzzy match
- 🟣 **Purple**: City-level fallback
